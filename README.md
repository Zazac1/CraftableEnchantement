# Craftable Enchantement

Mod Fabric pour Minecraft 1.21.11 qui introduit une progression d'enchantement craftable et lisible en survie. Cette base inclut un item d'exemple (`enchanting_essence`) craftable via table de craft, correctement enregistré avec `registryKey`, injecté dans le menu créatif via `ItemGroupEvents`, et prêt pour extension (nouvelles recettes, mécaniques et équilibrage).

## Stack

- Minecraft: 1.21.11
- Yarn: 1.21.11+build.6
- Fabric Loader: 0.19.3
- Loom: 1.17.13
- Fabric API: 0.141.4+1.21.11
- Mod Menu: 17.0.0
- Java: 21

## Build

```powershell
./gradlew build
```
